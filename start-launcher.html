Trump News Tracker - Launcher

This page will help you launch the Trump News Tracker application.

Unfortunately, browsers cannot directly run scripts for security reasons.

Please follow one of these methods to start the application:

Method 1: Use the VBS Launcher (Recommended for Windows)

Double-click the launch-app.vbs file in the same folder as this HTML file.

Method 2: Manual Command Line

  1. Open a command prompt or terminal in this folder
  2. Run one of the following commands depending on your operating system:

For Windows:

start-app.bat

For Mac/Linux:

./start-app.sh

After running the command, the application will start and open in your browser automatically.

(This button tries to use ActiveX which only works in Internet Explorer)
